# Projeto-de-Software-web
Link do repositório para o projeto de software da disciplina de programação de Web 2
O Sistema é uma aplicação web monolítica de gestão escolar desenvolvida para centralizar e integrar os processos administrativos e pedagógicos de uma escola de música.
O sistema reúne, em um único ambiente, o controle de alunos, responsáveis, professores, turmas, cursos, salas e contratos, garantindo que as informações circulem de
forma consistente entre os módulos e permaneçam acessíveis aos usuários autorizados.
